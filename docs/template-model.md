# Модель шаблонов и печатных форм

## Сущности
- `Template`: карточка шаблона (название, модуль, формат DOCX/XLSX, статус)
- `TemplateVersion`: конкретная версия файла шаблона
- `PlaceholderMapping`: соответствие плейсхолдера полю доменной модели
- `GeneratedDocument`: экземпляр, сформированный по карточке записи

## Жизненный цикл
1. Администратор создает шаблон и загружает файл версии.
2. Настраивает mapping плейсхолдеров (`{{ full_name }}`, `{{ position }}`, ...).
3. Отмечает версию активной.
4. Пользователь в карточке записи нажимает «Сформировать печатную форму».
5. Сервис генерации берет активную версию и контекст записи.
6. Генерируется docx/xlsx, файл сохраняется в архив.
7. Пользователь может повторно открыть и распечатать архивный экземпляр.

## MVP генерации Word
- Движок: `docxtpl`.
- Поддержка:
  - одиночных полей;
  - многострочных полей;
  - повторяемых блоков (`{% for row in items %}`) для списков обучения/стажировок/опыта.
